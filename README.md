# 📈 Trading Yield Simulator (JavaScript)

A lightweight simulation tool built in JavaScript to calculate **monthly compound interest** for trading accounts and evaluate target sustainability.

## 🚀 Key Features
- **Monthly Loop Simulator:** Uses a `for` loop to calculate and display capital growth month by month.
- **Risk Evaluation:** Built-in logic that warns the user if their monthly percentage target involves excessive risk.
- **Financial Math:** Applies percentage calculations directly into programming logic.

## 🛠️ Logic & Syntax Covered
- **Loops:** `for` loop for temporal simulations.
- **Conditionals:** `if/else` structures for risk guardrails.
- **Functions:** Modular code blocks with parameters and return values.
